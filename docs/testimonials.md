# Testimonial Management Plan

## Collection
- Request testimonial after Payment Complete + first milestone
- Provide structured prompts (problem, solution, outcome, metrics)
- Capture consent and usage rights

## Verification
- Confirm identity via project email or contract reference
- Validate claims (metrics, timelines) with internal records

## Review Schedule
- Quarterly review to update featured testimonials
- Rotate by industry and project type

## Approval Workflow
- Draft → Internal review → Client approval → Publish
- Track versions and approval timestamps
