# Client Self-Service Capabilities

## Guides
- Describe project → Analyze → Get Custom Price → Save/Pay
- Upload `.txt` SRS or paste description
- Choose experience level; review drivers and breakdown
- Proceed to payment; pick 50% deposit or full payment

## Step-by-Step
- Quote page: enter SRS → click "Analyze Requirements"
- Review detected drivers and base estimate
- Click "Get Custom Price" → view summary
- Click "Proceed to Payment" → checkout
- Complete billing and payment method; agree to terms

## Troubleshooting
- File not accepted: ensure `.txt` mime type; paste instead
- Empty analysis: add concrete requirements (auth, payments, admin)
- Card errors: check number (Luhn), expiry format MM/YY, CVV 3–4
- Coupon invalid: verify code spelling and availability
- No order found: ensure you came from a quote; URL includes `id`
